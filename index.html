Chúc mừng bạn đã đến với Khoa Công nghệ thông tin.

Trong niên khóa này tại đây chúng ta sẽ nghiên cứu môn "Đồ án lập trình hướng đối tượng".

Mô tả các nội dung môn học Đồ án cơ sở lập trình

Môn học nhằm nâng cao kỹ năng lập trình các bài toán liên quan đến vấn đề kỹ thuật lập trình hướng đối tượng.

Nhiệm vụ của sinh viên

  • Dự lớp: Tối thiểu có mặt 80% giờ học
  • Thực hành:Làm đầy đủ các bài tập thực hành được giao
  • Đọc tài liệu trước khi lên lớp
  • Đọc thêm tài liệu mà giáo viên yêu cầu

Tài liệu tham khảo

  • [1] Phạm Hồng Thái, Bài giảng ngôn ngữ lập trình C/C++, Trường Đại học Công nghệ, Đại học Quốc gia Hà Nội. Hà Nội 2003.
  • [2] Giáo trình C căn bản, HanoiAptech Computer Education Center (uet.vnu.edu.vn/tltk/File_PDF/Giao_Trinh_C_Can_Ban.pdf)
  • [3] Giáo Trình Bài Tập Kỹ Thuật Lập Trình, Trường CĐ Công Nghệ Thông Tin TP. Hồ Chí Minh

Tuần 1

Buổi 1 : Ôn tập lập trình hướng đối tượng

 

Bài tập: Cài đặt lớp NGUOI gồm

dữ liệu: họ và tên, giới tính, địa chỉ, tiền thưởng

phương thức: lĩnh thưởng.

Cài đặt lớp SV kế thừa lớp NGUOI và bổ sung

dữ liệu: điểm trung bình(DTB);

phương thức: định nghĩa lại phương thức lĩnh thưởng

để thêm điều kiện nếu DTB>8 thì được nhận 500000đ.

Viết chương trình nhập mảng SV có n người (n<10),

in danh sách các sinh viên lĩnh thưởng.

Hướng dẫn sử dụng C++ với winform cài đặt một ứng dụng đơn giản.

#include <iostream>

#include<fstream>

#include <string>

using namespace std;

 

class SV

{

string _maSV;

string _tenSV;

public:

SV() {};

SV(string a, string b) { _maSV = a; _tenSV = b; };

string maSV() { return _maSV; }

string tenSV() { return _tenSV; }

void maSV(string x) { _maSV = x; }

void tenSV(string x) { _tenSV = x; }

void printSV(ostream& out)

{

out << _maSV << " " << _tenSV << endl;

}

 

};

class LOP {

//Danh sach sinh vien toi da 100 nguoi

SV _a[100];

//So nguoi hien dang co

int _n;

public:

LOP() { _n = -1; };

int size() { return _n; }

void size(int x) { _n = x; }

SV a(int i) { return _a[i]; }

void a(int i, SV* x) { _a[i] = *x; }

void xoa(int i) {

if (i >= 0 && i <= _n) {

for (int j = i; j < _n; j++)_a[j] = _a[j + 1];

_n--;

}

}

void sua(int i, SV* x){

if (i >= 0 && i <= _n) _a[i] = *x;

}

void printLOP(char fileName[]) {

ofstream fout(fileName);

for (int i = 0; i<=_n; i++) { _a[i].printSV(fout); }

fout.close();

}

};
